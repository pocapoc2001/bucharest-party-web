How to run the app locally on your computer:

1. First, clone the repository by doing: git clone https://github.com/pocapoc2001/bucharest-party-web.git on your enviroment terminal
2. Second, change the directory to the folder of the project: cd bucharest-party-web
3. Install dependencies and all packets used in this project: npm install (this requires specific allowance, maybe run it with cmd before)
4. To run the app: npm run dev
